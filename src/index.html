How to play

  • Choose a size of field by clicking the blue button and selecting from the list.
  • You can move all tiles on the board in four directions: up, down, left, and right using arrow keys.
  • When two tiles with the same number collide after a move, they merge into a single tile with the sum of their values. For example, if two "2" tiles collide, they become a single "4" tile.
  • The goal of the game is to create a tile with the number 2048.
  • After each move, a new tile with a value of "2" or "4" appears randomly on an empty spot on the board
  • The game ends when the board is full, and no more moves are possible, or when you achieve a tile with the value of 2048.
